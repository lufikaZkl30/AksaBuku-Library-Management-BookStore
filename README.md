## Panduan run website:
- Buka di terminal
```bash
npm run start
```
- Buka web di 
```bash
localhost:8080
```

## Panduan database:
- Menggunakan Mongo DB
- Gunakan database mongodb://localhost:27017
- Import file [Database Buku](public/DatabaseBuku.csv) ke dalam database aksabuku, collection books
